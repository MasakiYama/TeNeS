***************************
使用方法
***************************

``TeNeS`` では事前に所定のフォーマットに従い模型や演算順などを定義するための複数の入力ファイルを作成する必要があります。
入力ファイルを直接作成し実行することも可能ですが、利用しやすいように、

- あらかじめ定義された格子模型に対して, 簡単な入力ファイルから ``tenes`` を実行するための入力ファイルを生成するスクリプト ``tenes_simple.py`` 、 

- 所定のフォーマットに従い、自分で格子模型などを定義した入力ファイルを作成し実行することで、 ``tenes`` を実行するための入力ファイルを生成するスクリプト ``tenes_standard.py`` (ver. 1.0 で実装予定)、

を用意しています。

以下、それぞれのスクリプトの使用方法について記載し、最後に ``tenes`` の使用方法について説明します。
``tenes`` 本体への入力ファイルの詳細については、ファイルフォーマットをご覧ください。
